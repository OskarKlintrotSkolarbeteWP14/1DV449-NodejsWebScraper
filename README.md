### Testa applikationen lokalt
Efter att ha klonat repot, installera beroendena:
```
cd <repo folder>
npm install
```

Nu kan du köra en lokal server med live preview:
```
npm start
```
Appen finns på http://localhost:3000

För att bygga en statisk sida kör:
```
npm run build
```
